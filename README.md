# Shells_Trainer

Trainers for Shell based activities

All docker compose based.  

In the relevant directory 

Start with 

```
docker-compose up
```

stop with 

```
docker-compose down
```

## Shells

Covers SSH and Telnet Etc
 
Run with 
 
```
cd shells
shells$ docker-compose up
```

## WebShells

Covers WEb Based shells

```
cd WebShellPlayground
docker-compose up
```
